# Excel для сериала: что заполнять, кто, в каком порядке

Основано на реальном шаблоне `templates/project_template_v8.xlsx`.  
Язык простой. Схемы — ниже.

---

## 1. Что есть в шаблоне сейчас (5 листов)

```text
project.xlsx
├── Общий план     ← смысл ролика / структура
├── план           ← кадры колонками (главный рабочий лист)
├── Фоны           ← места / фоны
├── Предметы       ← реквизит
└── Персонажи      ← герои
```

**Важно:** шаблон заточен под **короткий ролик с закадровым текстом** (строка 49 = закадр).  
Для полноценного сериала этого **мало**: нет диалогов, нет списка ракурсов как производства, нет сезона/серий, нет дневника непрерывности, нет звука/монтажа.

Ниже: как использовать текущие графы + какие листы добавить.

---

## 2. Лист «Общий план» — что зачем

| Ячейки / блок | Что писать | Кто заполняет |
|---------------|------------|---------------|
| R2 Главная тема | о чём серия/ролик | агент библии / плана |
| R3 Главная мысль | главный тезис | то же |
| R4 Смысл для зрителя | зачем смотреть | то же |
| R6–R14 Темы и подтемы | блоки смысла: название, подтемы, функция, роль | агент плана |
| R15–… Поэпизодный план | куски по времени: №, таймкод, объём знаков, что раскрыть, персонажи, предметы, особенности | агент плана серии |

**Для сериала:** этот лист = **план одной серии** (не всего сезона).  
Сезон лучше вынести на новый лист (см. §6).

## 3. Лист «план» — устройство

Каждый **столбец** (с C3) = один кусок производства.  
Сейчас в коротком ролике столбец ≈ «одна мысль закадра».  
В сериале столбец должен стать **одним ракурсом** (или одной сценой на раннем этапе — см. порядок).

### 3.1. Блоки «кадр1 / кадр2 / кадр3»

В шаблоне уже заложены **до 3 ракурсов на сцену**:

| Блок | Строки | Смысл |
|------|--------|--------|
| кадр1 | R2–R14 | первый ракурс сцены |
| кадр2 | R17–R29 | второй ракурс |
| кадр3 | R32–R44 | третий ракурс |

Сейчас агенты часто заполняют **только кадр1**.  
Для сериала нужно заполнять **кадр1+кадр2+кадр3** (и при необходимости больше столбцов = больше ракурсов).

### 3.2. Графы одного ракурса (повторяются в каждом блоке)

| Строка | Графа | Что писать | Кто |
|--------|-------|------------|-----|
| id scene | id сцены | `scene_01` | агент разбора сцен |
| id shot | id ракурса | `shot_01` / `shot_02`… | агент ракурсов |
| номер кадра | полный id | `scene_01_shot_01` | то же |
| предыдущий / следующий кадр | связь | куда клеить | агент ракурсов / монтажа |
| фон | id фона | `f01` с листа Фоны | агент разбора / фонов |
| персонажи | кто в кадре | `c01,c02` | агент разбора / ракурсов |
| предметы | что видно | id с Предметы | то же |
| действие | что делают | видимое действие | агент ракурсов |
| описание кадра | картинка словами | композиция | агент ракурсов |
| логика перехода | зачем резка | «реакция на фразу» | агент ракурсов / монтажа |
| заметки по консистенции | не ломать | одежда, рана, день | агент непрерывности |
| статус | готовность | пусто / ок / переделать | человек или контроль |

### 3.3. Общие строки столбца (под всеми блоками кадров)

| Строка | Графа | Сейчас (short) | Для сериала | Кто |
|--------|-------|----------------|-------------|-----|
| **R45** | промт картинки 1 | промт shot1 | промт ракурса 1 | агент промтов картинок |
| **R46** | промт картинки 2 | редко | промт ракурса 2 | то же |
| **R47** | промт картинки 3 | редко | промт ракурса 3 | то же |
| **R48** | промт видео | анимация shot1 | видео ракурса 1 | агент анимации |
| **R49** | закадровый текст | главный источник смысла | **не главный**; см. лист Диалоги | агент сценария (только если нужен рассказчик) |
| **R50** | время на кадр | сек | длительность ракурса | разбор / монтаж |
| **R52** | главное действие | коротко «что рисуем» | действие ракурса / сцены | агент разбора / ракурсов |
| **R54** | Место | дом, улица… | = локация сцены | агент разбора |
| **R55** | Акцент | на общении / предмете… | тип акцента камеры | агент ракурсов |
| **R56** | Смысл сцены | зачем сцена | цель сцены | агент плана / разбора |
| **R57** | Тип сцены | реализм… | + тип покрытия: диалог / давление / идут… | агент ракурсов |
| **R58** | Особенность | эмоции, пейзаж… | пометки | разбор |
| **R59** | Номер кластера | группа | можно = номер серии/акта | план |
| **R64** | промт видео 2 | shot2 video | видео ракурса 2 | агент анимации |

## 4. Листы Фоны / Предметы / Персонажи

### Персонажи (столбцы c01, c02…)
Заполняет **агент героев** (после библии сезона):

- имя, внешность, одежда, характер, правила  
- R20 финальный промт персонажа → генерация референса  

**Добавить для сериала (на этом же листе или новом):**  
голос (id озвучки), роль (главный/второй), состояние по сериям.

### Фоны (столбцы f01…)
Заполняет **агент мест**:

- место, эпоха, атмосфера, свет, палитра, планы, камера фона  
- итоговый / финальный промт фона  

### Предметы
Заполняет **агент предметов / разбор сценария**:

- id, название, описание, постоянный ли, когда появляется, у кого, правила

## 5. Чего не хватает для сериала → новые листы

Текущих 5 листов **недостаточно**. Нужно добавить:

```text
НОВЫЕ ЛИСТЫ (предложение)
├── Сезон              ← библия + карта серий
├── Серия              ← паспорт одной серии (можно 1 лист на файл серии)
├── Сценарий           ← сцены + реплики (вместо опоры на R49)
├── Ракурсы            ← полный shot list (если «план» перегружен)
├── Непрерывность      ← дневник фактов
├── Звук               ← речь / музыка / шумы по меткам
└── Монтаж             ← порядок склейки + замок картинки
```

Минимальный набор новых: **Сезон, Сценарий, Непрерывность, Звук**.  
Ракурсы можно сначала вести в блоках кадр1/2/3 листа «план».

## 6. Новые листы — какие графы

### 6.1. Лист «Сезон»
| Графа | Кто пишет |
|-------|-----------|
| Название, логлайн | человек / агент библии |
| Тон, мир, правила | агент библии |
| Список серий (E01…): о чём, крючок | агент карты сезона |
| Арки героев на сезон | агент карты сезона |
| Запреты / канон | библия + дневник |

### 6.2. Лист «Сценарий»
Одна строка = одна реплика или одна ремарка.

| Графа | Пример | Кто |
|-------|--------|-----|
| № серии | E01 | — |
| № сцены | S03 | агент разбора |
| Место | ВНУТРИ. КУХНЯ | сценарий |
| День/ночь | НОЧЬ | сценарий |
| Цель сцены | уличать во лжи | план/сценарий |
| Поворот сцены | признаётся | план/сценарий |
| Тип строки | реплика / действие | сценарий |
| Герой | c02 Маша | сценарий |
| Текст | «Ты снова врёшь» | агент диалогов |
| Эмоция | тише, злость | правка речи |
| К какому ракурсу | scene_03_shot_02 | агент ракурсов |

**Это замена главной роли строки R49** для сериала.

### 6.3. Лист «Непрерывность»
| Графа | Кто |
|-------|-----|
| Факт | агент непрерывности |
| С какой серии/сцены действует | то же |
| Тип: сюжет / одежда / рана / знание | то же |
| Статус: активно / закрыто | то же |

### 6.4. Лист «Звук»
| Графа | Кто |
|-------|-----|
| Метка времени / сцена / ракурс | разметка звука |
| Тип: речь / музыка / шум / тишина | то же |
| Герой или кусок музыки | озвучка / музыка |
| Файл/статус | генераторы звука |

### 6.5. Лист «Монтаж» (можно позже)
Порядок: scene_shot → длительность факт → статус замка картинки.

## 7. Кто какой агент и куда пишет

```text
АГЕНТЫ                          КУДА ПИШУТ РЕЗУЛЬТАТ
─────────────────────────────────────────────────────
Библия сезона                →  лист «Сезон»
Карта сезона                 →  лист «Сезон» (таблица серий)
План серии                   →  «Общий план» (+ цель/поворот в «Сценарий»)
Сценарий + диалоги           →  лист «Сценарий»
Правка речи                  →  тот же «Сценарий» (эмоция/текст)
Разбор сцен                  →  «Сценарий» (цели) + старт столбцов «план»
Герои                        →  «Персонажи»
Места/фоны                   →  «Фоны»
Предметы                     →  «Предметы»
Список ракурсов              →  блоки кадр1/2/3 на «план» (+ id в «Сценарий»)
Непрерывность                →  лист «Непрерывность» + R13 на «план»
Промты картинок              →  R45/R46/R47
Промты видео                 →  R48/R64
Озвучка диалогов             →  лист «Звук» (+ файлы на диске)
Музыка/шумы                  →  лист «Звук»
Монтаж                       →  лист «Монтаж» / статус на «план»
Человек (утверждение)        →  статус / замок
```

## 9. Как читать лист «план» в режиме сериала

### Вариант A (рекомендую на старте) — столбец = сцена, внутри 3 ракурса

```text
Столбец D = сцена 1
  кадр1 (R2–14)  = общий план
  кадр2 (R17–29) = через плечо А
  кадр3 (R32–44) = через плечо Б / реакция
  R45/R46/R47    = промты трёх картинок
  R48/R64        = видео 1 и 2 (третье — позже отдельной графой)
  R49            = пусто ИЛИ короткий рассказчик (не диалог!)
  Диалоги        = только лист «Сценарий», привязка к scene/shot
```

Если ракурсов нужно больше 3 — **новый столбец** с тем же `id scene`, следующие `shot_04+`  
или расширить шаблон блоками кадр4/кадр5.

### Вариант B — столбец = один ракурс

Проще для генераторов, но лист становится очень широким.  
Тогда id scene повторяется в соседних столбцах.

Для сериала с диалогами лучше **A + лист Сценарий**.

## 10. Что заполняет человек руками

Обычно только:

1. Тема/бриф сезона (старт)  
2. Утверждения (ок / переделать) на каждом стоп-кране  
3. Вычёркивание лишних ракурсов  
4. Редкие правки текста диалога  

Всё остальное пишут агенты в листы.

---

## 11. Схема «кто за кем» одной фразой

```text
Сезон → План серии → Сценарий/диалоги → Факты
      → Герои/Фоны/Предметы
      → Разбор сцен → Ракурсы
      → Промты картинок → Картинки
      → Промты видео → Видео
      → Речь/музыка/шумы
      → Монтаж → Замок → Финал → Запись в сезон
```

Без утверждённого сценария ракурсы не начинать.  
Без ракурсов не писать R45–R48.  
Без замка картинки не финалить звук.

## 12. Соответствие старому short-пайплайну

| Было (short) | Станет (сериал) |
|--------------|-----------------|
| Общий план = план ролика | Общий план = план серии |
| R49 закадр = смысл | Сценарий = смысл; R49 почти не главный |
| один shot на столбец | кадр1/2/3 = coverage |
| enrich заполняет R2–14 от R49 | разбор+ракурсы от листа Сценарий |
| audio = один голос | Звук = много голосов по репликам |
| нет сезона | лист Сезон |
| нет дневника | лист Непрерывность |

## 13. Практический вывод

1. **Текущий шаблон уже почти готов как лист раскадровки** (план + 3 кадра + фоны/герои/предметы).  
2. **Для сериала обязательно добавить:** Сезон, Сценарий, Непрерывность, Звук.  
3. **Главная смена логики:** смысл берём из **Сценария (диалоги)**, а не из R49.  
4. **Камера:** заполнять кадр1/2/3 по рецептам из `SERIES_CAMERA_ANGLES.md`.  
5. **Порядок агентов** — §8; места записи — §7.
