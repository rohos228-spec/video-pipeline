## Как использовать этот отчёт

1. Закрыть PR'ы с `close-as-superseded` массово через GitHub UI.

3. `rebase-or-close` — попросить автора rebase'ить или закрыть.
4. `review` — посмотреть руками, решить.
5. Ветки `cursor/audit-*` — удалить массово (если не нужны cherry-pick'и).
