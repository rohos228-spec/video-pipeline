# Шпаргалка: как работать с сериальными агентами

Простым языком. Для людей, которые запускают этапы вручную в ChatGPT / Studio.

**Файлы:**
- Excel: `templates/project_template_series_v1.xlsx`
- Промпты агентов: `prompts/steps/series/ser_XX_*/agent.txt` (каждый ≥ 10 000 символов)
- Камера: `docs/SERIES_CAMERA_ANGLES.md`

---

## 1. С чего начать (один раз)

1. Скопируй шаблон → рабочий файл серии, например:  
   `data/series/my_show/season_01/e01/project.xlsx`
2. Открой лист **Сезон**.
3. В чат с GPT вставь **весь** файл  
   `prompts/steps/series/ser_01_bible/agent.txt`
4. Ниже вставь вводные (см. §3).
5. Прикрепи Excel (или попроси заполнить поля текстом и перенеси сам).
6. Проверь результат → напиши **«утверждаю»** → только потом следующий агент.

**Правило:** пока не утвердил этап — следующий агент не запускай.

## 2. Куда что писать (карта)

| Что нужно получить | Куда смотреть в Excel | Какой агент |
|--------------------|----------------------|-------------|
| Мир, тон, запреты | Сезон C3–C10 | ser_01_bible |
| Список серий, крючки, арки | Сезон таблица серий + арки | ser_02_season_map |
| План одной серии с поворотами | Общий план | ser_03_episode_outline |
| Диалоги и действия | **Сценарий** | ser_04_teleplay |
| Более живая речь | Сценарий (текст/эмоция) | ser_05_dialogue_polish |
| Факты «нельзя ломать» | Непрерывность | ser_06_continuity |
| Лица, одежда, голоса | Персонажи | ser_07_characters |
| Места | Фоны | ser_08_locations |
| Важные вещи | Предметы | ser_09_items |
| Сцены столбцами | план (разбор) | ser_10_breakdown |
| Ракурсы камеры | план кадр1/2/3 + R60–R63 | ser_11_coverage |
| Промты картинок | план R45–R47 | ser_12_image_prompts |
| Картинки файлами | папка + R66 | ser_images |
| Промты видео | план R48/R64/R65 | ser_13_anim_prompts |
| Видео файлами | папка + R66 | ser_videos |
| Речь героев | Звук (тип=речь) | ser_14_dialogue_audio |
| Музыка/шумы | Звук | ser_15_music_sfx |
| Порядок склейки | Монтаж | ser_16_assemble |
| Фиксация нарезки | Монтаж замок + R67 | ser_17_picture_lock |
| Финальный звук | файл + Звук | ser_18_final_mix |
| Проверка | отчёт | ser_20_qc |
| Выпуск | Сезон статус + факты | ser_19_deliver |

**Диалоги никогда не пишутся в строку 49 листа «план».**  
Строка 49 — только если нужен редкий рассказчик. Обычные реплики — лист **Сценарий**.

## 3. Какие вводные давать на каждом этапе

Копируй блок целиком. В квадратных скобках — твоё.

### Этап 1 — Библия (`ser_01_bible/agent.txt`)

```text
ВВОДНЫЕ:
Название: [название]
Жанр/тон: [например мрачная семейная драма]
Число серий: [5]
Длина серии: [8 минут]
Формат картинки: [16:9]
Главные герои (имена): [Маша, Иван, Олег]
Логлайн своими словами: [...]
Чего нельзя ломать: [например: Маша не убийца; город один и тот же]
Дополнительно: [...]
```

### Этап 2 — Карта сезона (`ser_02_season_map`)

```text
ВВОДНЫЕ:
Библия на листе «Сезон» уже утверждена.
Сделай карту на [5] серий и арки героев.
Особый акцент: [например серия 3 — поворот про письмо]
```

### Этап 3 — План серии (`ser_03_episode_outline`)

```text
ВВОДНЫЕ:
Номер серии: [E01]
Крючок предыдущей серии: [нет / текст]
Работай только с этой серией.
Заполни лист «Общий план»: завязка + 3 части с поворотами + концовка.
```

### Этап 4 — Сценарий (`ser_04_teleplay`)

```text
ВВОДНЫЕ:
Серия: [E01]
План на листе «Общий план» утверждён.
Заполни лист «Сценарий».
Диалоги НЕ писать в строку 49 листа «план».
```

### Этап 5 — Правка речи (`ser_05_dialogue_polish`)

```text
ВВОДНЫЕ:
Серия: [E01]
Поправь только реплики и эмоции на листе «Сценарий».
Новых фактов сюжета не добавлять.
```

### Этап 6 — Непрерывность (`ser_06_continuity`)

```text
ВВОДНЫЕ:
Серия: [E01]
Сценарий утверждён.
Обнови лист «Непрерывность».
Если конфликт с каноном — напиши БЛОК.
```

### Этапы 7–9 — Герои / Фоны / Предметы

```text
ВВОДНЫЕ:
Серия: [E01]
Сценарий утверждён.
Заполни лист [Персонажи / Фоны / Предметы] по контракту агента.
```

Можно параллельно 7, 8, 9 после этапа 6 (если нет БЛОКа).

### Этап 10 — Разбор (`ser_10_breakdown`)

```text
ВВОДНЫЕ:
Серия: [E01]
Справочники готовы.
Создай столбцы сцен на листе «план».
Не трогай R45–R48 и не пиши диалоги в R49.
```

### Этап 11 — Ракурсы (`ser_11_coverage`)

```text
ВВОДНЫЕ:
Серия: [E01]
Разбор готов.
Заполни кадр1/2/3 и R60–R63.
Обнови привязку реплик в «Сценарий».
После ответа я вычеркну лишние ракурсы.
```

### Этап 12 — Промты картинок → картинки

```text
ВВОДНЫЕ:
Ракурсы утверждены (лишнее вычеркнул).
Заполни R45–R47.
```

Потом runbook/agent `ser_images` (генерация файлов) → ты смотришь галерею.

### Этап 13–15 — Видео-промты → видео

Аналогично: `ser_13_anim_prompts` → `ser_videos`.

### Этап 16 — Озвучка

```text
ВВОДНЫЕ:
Серия: [E01]
Сценарий утверждён, голоса на Персонажах заданы.
Заполни лист «Звук» строками тип=речь.
```

### Этап 17 — Музыка/шумы (`ser_15_music_sfx`)

После чернового монтажа или хотя бы после сценария; финальный микс — только после замка.

### Этап 18 — Монтаж (`ser_16_assemble`)

```text
ВВОДНЫЕ:
Серия: [E01]
Видео и речь готовы.
Собери лист «Монтаж». Замок не ставить.
```

### Этап 19 — Замок (`ser_17_picture_lock`)

```text
ВВОДНЫЕ:
Я утверждаю нарезку. Проведи чеклист и поставь замок, если всё ок.
```

### Этап 20–22 — Микс → QC → выпуск

Только при `замок=да`.

## 4. Порядок (не нарушать)

```text
1 библия ──утвердил──► 2 карта сезона ──утвердил──►
3 план серии ──утвердил──► 4 сценарий ──► 5 правка речи ──утвердил──►
6 непрерывность ──(если не БЛОК)──►
7 герои + 8 фоны + 9 предметы ──утвердил главных──►
10 разбор ──► 11 ракурсы ──вычеркнул лишнее──►
12 промты картинок ──► картинки ──утвердил──►
13 промты видео ──► видео ──утвердил──►
14 озвучка ──► 15 музыка/шумы
16 монтаж ──утвердил──► 17 ЗАМОК
18 финальный звук ──► 20 приёмка ──► 19 выпуск
```

## 5. Как вставлять промпт агента

1. Открой `agent.txt` нужного этапа.  
2. Скопируй **весь** файл в системный промпт / первое сообщение.  
3. Вторым сообщением — блок **ВВОДНЫЕ** из §3.  
4. Прикрепи актуальный Excel.  
5. Проси: «Верни обновлённый Excel» или таблицу изменений по контракту.  
6. Сам перенеси в файл, если модель не умеет править xlsx напрямую.

В Studio: если этап подключён к ноде — в ноду кладётся именно `agent.txt`, не короткий `template.md`.

## 6. Частые ошибки

| Ошибка | Как правильно |
|--------|----------------|
| Диалоги в R49 | Только лист Сценарий |
| Запустил ракурсы до сценария | Сначала 4→5→6 |
| Финальный звук до замка | Сначала 17 замок |
| Один ракурс на весь диалог | Минимум общий + 2 встречных + реакция |
| Переписал библию на этапе 4 | Библию трогает только этап 1 (и ты руками) |
| Не вычеркнул лишние ракурсы | После 11 обязательно руками |
| Забыл обновить Непрерывность после выпуска | Этап 19 |

## 7. Мини-чеклист одной серии

- [ ] Сезон: библия + карта утверждены  
- [ ] Общий план серии утверждён  
- [ ] Сценарий утверждён, диалоги не в R49  
- [ ] Непрерывность без БЛОКа  
- [ ] Персонажи/Фоны/Предметы ок  
- [ ] план: сцены + ракурсы утверждены  
- [ ] R45–47 → картинки ок  
- [ ] R48/64/65 → видео ок  
- [ ] Звук: речь по героям  
- [ ] Монтаж собран  
- [ ] Замок = да  
- [ ] Финальный звук  
- [ ] QC ок  
- [ ] Выпуск + факты в Сезон/Непрерывность
