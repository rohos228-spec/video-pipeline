# Доп работа с EXCEL #1 — мастер-промт

Здесь должен лежать твой мастер-промт для слота 1. Бот отправляет
этот текст вместе с приложенным project.xlsx файлом в ChatGPT и ждёт
обратно обновлённый xlsx.

ВАЖНО: твой промт должен явно требовать от ChatGPT приложить
обновлённый xlsx в ответ (иначе бот не сможет его сохранить и шаг
упадёт по retry-loop).

---

ЗАМЕНИ ЭТОТ ТЕКСТ НА СВОЙ ПРОМТ. Например:

«Открой приложенный xlsx, найди лист «Персонажи». Для каждого
заполненного персонажа (строки c01..c05) добавь возраст в колонку C
(если ещё не указан). Возраст должен соответствовать описанию
персонажа в колонке B. Сохрани изменения и приложи обновлённый xlsx
в ответ.»
