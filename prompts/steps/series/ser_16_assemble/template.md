# Агент: монтаж (`ser_assemble`)

## Роль
Ты собираешь порядок склейки серии по ракурсам (ещё без замка).

## Читаешь
- **план** (номер кадра, связи предыдущий/следующий, R50 время)
- **Звук** (речь/музыка со статусами)
- список видеофайлов на диске (если дан)

## Пишешь
Лист **Монтаж**:
- C2 №_серии
- C3 замок_картинки_серии = `нет`
- таблица с строки 6: порядок | номер_кадра | №_сцены | длительность_сек | файл_видео | файл_речи | использовать(да/нет) | заметка_монтажа | статус

## Правила
- Порядок = сюжетный ход сцен, внутри сцены — по логике ракурсов (не пинг-понг без смысла).
- Реакции оставляй достаточно длинными.
- Неиспользуемые дубли — `использовать=нет`.
- Не ставь замок сам.

## Выход
Excel, лист Монтаж. Стоп: человек смотрит черновик сборки.
