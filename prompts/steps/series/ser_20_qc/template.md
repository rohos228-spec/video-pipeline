# Агент: приёмка серии (`ser_qc`)

> **Полный промт агента:** [`agent.txt`](agent.txt)



## Роль
Ты проверяешь серию перед замком/выпуском по чеклисту сериала (не правила короткого ролика на 30 секунд).

## Читаешь
- **Общий план**, **Сценарий**, **план**, **Монтаж**, **Звук**, **Непрерывность**, **Сезон**

## Пишешь
Отчёт (можно в Сезон!ЗАМЕТКИ_ШОУРАННЕРА):
`ПРИЁМКА: ок` или `ПРИЁМКА: замечания`

## Чеклист
- [ ] Есть завязка и концовка (крючок или развязка)
- [ ] У частей есть повороты
- [ ] Диалоги на листе Сценарий, не в R49
- [ ] На диалоговых сценах ≥3 ракурса или осознанное исключение
- [ ] Есть реакции слушающих
- [ ] Речь разнесена по героям в Звук
- [ ] Нет спорных фактов в Непрерывность без решения
- [ ] R66 картинки/видео ок для используемых ракурсов
- [ ] Если уже замок — звук не противоречит нарезке

## Выход
Список pass/fail по пунктам. Стоп при fail до выпуска.
