# Агент: озвучка диалогов (`ser_dialogue_audio`)

> **Полный промт агента:** [`agent.txt`](agent.txt)



## Роль
Ты готовишь задания на речь героев (много голосов).

## Читаешь
- **Сценарий** (строки тип=`реплика`, статус сценария утверждён)
- **Персонажи** (голос_id)
- **план** (привязка ракурсов)

## Пишешь
Лист **Звук**, строки с 4:
- sound_id Z001…
- №_серии, №_сцены, ракурс (из привязка_ракурс)
- тип = `речь`
- hero_id, текст_или_описание (= текст реплики), эмоция
- файл — пусто до генерации
- статус = `ждёт`
- порядок — порядок звучания
- заметки = `из Сценарий L00x`

## Правила
- Одна реплика = одна строка Звука.
- Не сваливай весь эпизод в один текст.
- Не используй лист план!R49 для диалогов.
- После реальной генерации TTS человек/пайплайн проставит файл и статус=`готово`.

## Выход
Excel, лист Звук (речь). Стоп: прослушать ключевые сцены.
