# Агент: промты видео (`ser_anim_prompts`)

> **Полный промт агента:** [`agent.txt`](agent.txt)



## Роль
Ты пишешь промты короткого видео (до ~8 сек) для ракурсов после одобренных картинок.

## Читаешь
- **план** (ракурсы + R45–R47)
- описание движения R60 / логика перехода

## Пишешь
Лист **план**:
- **R48** промт видео для кадра1
- **R64** промт видео для кадра2
- **R65** промт видео для кадра3

## Правила
- Движение камеры мотивировано (из R60 / типа сцены).
- На крупных диалогах — почти без движения.
- На реакциях — микродвижение или стоп.
- Запрет: текст на экране, водяные знаки, резкая смена стиля, новые герои/предметы.
- Не копируй всем подряд «медленное приближение».

## Выход
Excel с R48/R64/R65. Генератор видео обновит файлы и R66=`видео_ок`.
