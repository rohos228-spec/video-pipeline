ОБЯЗАТЕЛЬНЫЕ ЛИСТЫ:
  • «план» — главный источник.
    - R60 «движение камеры» — общая логика движения для сцены
    - R61 «крупность» — тип кадра (общий, через плечо, крупный...)
    - R62 «зачем этот ракурс» — мотивация (давление, реакция, переход)
    - R57 «тип сцены» — диалог / давление / экшн / идут_и_говорят / нарезка
    - R50 «время на кадр» — целевая длительность в секундах (≤8)
    - кадр1/кадр2/кадр3 — действие, описание кадра, персонажи
    - R45/R46/R47 — итоговые промты картинок (референс для консистентности)
    - R66 — статус генерации (пишем видео только для «картинки_ок»)

  • «Сценарий» — реплики сцены: кто говорит, кто молчит, какая эмоция.
    Это определяет движение персонажей (говорящий активнее, слушающий статичнее).
    Привязка R63 (line_id) — связывает ракурс с репликами.

  • «Фоны» — есть ли в фоне элементы, которые могут двигаться
    (листья дерева, занавеска от окна, телевизор с изображением).

ДОПОЛНИТЕЛЬНО:
  • «Непрерывность» — активные факты об эмоциональном состоянии, усталости,
    ранениях (влияет на то, как двигается персонаж).

НЕ ЧИТАЕШЬ:
  • R49 — закадровый текст short-пайплайна, не используется.
  • Монтаж, Звук — не твои источники на этом шаге.

§5. ПРАВИЛА НАПИСАНИЯ ПРОМТА ВИДЕО
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

5.1. ДЛИТЕЛЬНОСТЬ КЛИПА ≈ 8 СЕК

  Каждый клип — один ракурс, ≤8 секунд. Это не значит «много всего».
  8 секунд — это:
  - одна небольшая реакция
  - один жест с завершением
  - одно короткое движение камеры
  - статичный кадр с дыханием и мимикой

  Не пытайся вместить целую сцену в один клип. Не описывай «сначала → потом → потом».

5.2. ДВИЖЕНИЕ КАМЕРЫ — ТОЛЬКО МОТИВИРОВАННОЕ

  Разрешённые движения и когда применять:

  SLOW PUSH-IN (медленное приближение):
    → Когда: персонаж говорит важное, нарастает напряжение, ударная фраза.
    → Скорость: очень медленно, почти незаметно.
    → Запрет: использовать как дефолт для всех клипов.
    → Если больше 30% клипов серии — slow push-in → СТОП, выбери другие варианты.

  STATIC (полностью статичная камера):
    → Когда: крупный план слушателя, кадр реакции, момент тишины.
    → Разрешено и поощряется. Статика создаёт напряжение не хуже движения.

  VERY SLOW PAN (едва заметная горизонтальная проводка):
    → Когда: персонаж идёт в кадре, взгляд ведёт за движением.
    → Скорость: минимальная, не кинетичная.

  SLOW PULL-OUT (медленное отдаление):
    → Когда: финал сцены, персонаж остаётся один, ощущение одиночества.
    → Редко. Не в середине сцены.

  SLIGHT HANDHELD WOBBLE (лёгкое живое дрожание):
    → Когда: экшн, тревога, преследование, драка.
    → Только при типе сцены «экшн» или «давление». Не в спокойных разговорах.

  TILT DOWN / TILT UP:
    → Когда: раскрытие персонажа (начать с ног → поднять до лица или наоборот).
    → Редко. Только если это заложено в описании кадра.

  ЗАПРЕЩЁННЫЕ ДВИЖЕНИЯ:
    × Резкий zoom (хаотичный)
    × Rotation (вращение) без сильного обоснования
    × Quick cut (это монтаж, не камера)
    × Любое упоминание «smash cut», «match cut» в промте видео

5.3. ДВИЖЕНИЕ ПЕРСОНАЖЕЙ

  Описывай только то движение, которое логично следует из:
  - действия в блоке кадра (кадр1/2/3 на листе «план»)
  - реплики в Сценарии (говорящий vs слушающий)

  Типичные правила:
  - ГОВОРЯЩИЙ: жестикуляция, движение рта, смещение веса.
    Не «активно жестикулирует» — конкретно: «lifts one hand, palm up».
  - СЛУШАЮЩИЙ: почти стоп, микромимика, моргание, едва заметный наклон.
    Не «реагирует» — конкретно: «eyes shift slightly, jaw tightens».
  - ДИАЛОГОВЫЙ КАДР (КРУПНЫЙ): минимум движения тела, максимум мимики.
  - ОБЩИЙ ПЛАН: персонажи могут переместиться на шаг, повернуться.

5.4. СТРУКТУРА ПРОМТА ВИДЕО

  Порядок элементов:
  [ТИП ДВИЖЕНИЯ КАМЕРЫ, если есть] + [ЧТО ДЕЛАЕТ ПЕРСОНАЖ] +
  [ДЕТАЛИ ФОНА/ОКРУЖЕНИЯ, если двигаются] + [АТМОСФЕРА КЛИПА] +
  [ДЛИТЕЛЬНОСТЬ: ~Xs] + [ТЕХНИЧЕСКИЕ ТЕГИ]

  Пример:
  «Static camera. Masha slowly lifts her gaze from the table, mouth barely
  parting, as if about to speak but stopping herself. Her breath is slightly
  visible — cold room. Kitchen background steady, faint flicker of fluorescent
  light. Tense stillness. ~6s. No text, no watermarks, consistent with ref image.»

5.5. КОНСИСТЕНТНОСТЬ С КАРТИНКОЙ (R45/R46/R47)

  Видео-клип начинается с того же кадра, что референсная картинка.
  В промте НЕ описывай начальное состояние заново — генератор берёт его
  из картинки. Описывай только ИЗМЕНЕНИЕ относительно картинки.

  Запрещено в R48 описывать то, что противоречит R45:
  × В R45: «static, evening, Masha at sink» → в R48 нельзя «it's daytime»
  × В R45: «wide shot, both characters» → в R48 нельзя «close-up only»

5.6. ТИП СЦЕНЫ → ВЫБОР ДВИЖЕНИЯ

  Читай R57 «тип сцены»:

  «диалог» (обычный разговор):
    кадр1 (общий): slight handheld feel или static
    кадр2 (через плечо): static или едва заметный slow push
    кадр3 (реакция): ТОЛЬКО static — не давай реакции движение камеры

  «давление» (конфликт, давление, угроза):
    кадр2/кадр3: slow push-in → нарастание давления
    реакции: static с micro-expressions

  «идут_и_говорят» (ходьба):
    general: very slow pan следит за персонажами
    крупные вставки: static или minimal tilt

  «экшн» (погоня, драка, побег):
    slight handheld wobble разрешён
    быстрее движение персонажей, меньше камерного движения

  «нарезка_времени» (вставки):
    static, показывает деталь/объект
    может быть микро-фокус (не zoom, а rack focus)

5.7. ЧТО НЕЛЬЗЯ ПИСАТЬ В ПРОМТЕ ВИДЕО

  ✗ Текст/надписи в кадре
  ✗ Новые персонажи, которых не было в картинке
  ✗ Смена фона или локации внутри клипа
  ✗ Резкая смена стиля (не меняй визуальный язык)
  ✗ Диалоговые реплики («she says», «he shouts») — звук идёт отдельно
  ✗ Описание следующего кадра («then cut to...») — это монтаж, не клип
  ✗ «slow push everywhere» — мотивируй движение

§9. ТИПИЧНЫЕ ОШИБКИ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ОШИБКА 1: «Slow push in» для каждого кадра серии
  → Это дефолт, который убивает ритм. Используй static, pull-out, pan —
    разнообразь. Не более ~30% клипов с push.

ОШИБКА 2: Описываешь целую сцену в одном R48
  → «Masha enters, sits, talks, leaves» — это 4 клипа. R48 = только этот ракурс, ≤8 сек.

ОШИБКА 3: Движение камеры для реакционного крупного плана
  → Реакция = статичный кадр. Любое движение камеры уничтожает её вес.

ОШИБКА 4: Противоречие картинке
  → R45 говорит «interior, night» → R48 нельзя «daylight exterior».
    Видео начинается с кадра картинки.

ОШИБКА 5: Описание звука в промте видео
  → «she whispers», «loud crash» — звук идёт отдельно через Звук и озвучку.
    В промте видео — только визуальное движение.

ОШИБКА 6: Написан промт для кадра с R66 ≠ «картинки_ок»
  → Если картинка ещё не одобрена — видео не пишем. Ждём галерею.

§12. РАСШИРЕННЫЕ ПРАВИЛА ДВИЖЕНИЯ ПО ТИПУ РАКУРСА
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Ниже — детальная матрица «тип ракурса × движение камеры»:

ОБЩИЙ ПЛАН МЕСТА (Wide shot, establishing):
  → static или very slow pan если персонаж входит/выходит
  → не давай push-in на общий план места в начале эпизода
  → атмосферные детали (ветер, флаг, листья) можно упомянуть

ОБЩИЙ ПЛАН СЦЕНЫ (Wide shot, both characters):
  → Если оба стоят и говорят: slight camera drift (barely perceptible) или static
  → Если один подходит к другому: камера едва сопровождает движение
  → Не более 3 сек активного движения персонажа на 8-секундный клип

ЧЕРЕЗ ПЛЕЧО (Over-the-shoulder):
  → Почти всегда static или slow push-in (мотивация: нарастание диалога)
  → Говорящий: жест рукой, движение губ (не опиши сами слова)
  → Слушатель (затылок/плечо): не движется, якорит кадр

ЧИСТЫЙ ПОРТРЕТ / КРУПНЫЙ (Close-up):
  → По умолчанию: static
  → Если кульминация: очень медленный slow push-in (финальная треть клипа)
  → Не используй движение для портрета в середине диалога — только в пике

РЕАКЦИЯ (Reaction shot):
  → ВСЕГДА static. Это правило без исключений.
  → Описывай только мимику: «eyes narrow», «lip tightens», «gaze drops»
  → Никакого движения камеры, никакого движения тела (максимум — моргание)

ВСТАВКА / ДЕТАЛЬ (Insert shot):
  → static или очень медленный rack focus (от фона к предмету)
  → длительность обычно 3–5 сек
  → не превышай 6 сек для вставки

ХОДЬБА (characters walking):
  → very slow pan следит за движением
  → не позволяй персонажу выйти из кадра (если только это не сцена ухода)
  → лёгкое дыхание кадра (gentle sway) разрешено
