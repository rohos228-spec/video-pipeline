16/9 Create a dense professional character model sheet / turnaround sheet for the SAME exact character.

Character:
[age], [gender], [body type], [skin tone], [hair color and hairstyle], [facial features], [expression], [other defining traits]

Outfit:
[describe jacket / coat / top / pants / shoes / colors / materials]

IMPORTANT LAYOUT REQUIREMENTS:
pure white background (#FFFFFF)
no environment
no background shadows
no decorative elements
no text
no labels
no watermark

The sheet must be tightly packed and use almost all available canvas space.
No large empty areas.
Minimal margins.
Minimal spacing between views.
All views must be large and readable.
This must look like a dense packed professional reference sheet, not a poster.

COMPOSITION:

LEFT SIDE:
The left side of the sheet must contain 4 large full-body views of the SAME exact character.
Each figure should be almost as tall as the full canvas height.
Arrange them side by side from left to right:
1. full body front view
2. full body 3/4 front view
3. full body side view
4. full body back view

RIGHT UPPER AREA:
Show separate enlarged head turnaround views of the SAME character:
5. head front view
6. head 3/4 front view
7. head side profile
8. head back view
9. head 3/4 back view

RIGHT LOWER AREA:
Instead of large face close-ups, show separate enlarged clothing and footwear item studies:
10. [outerwear study]
11. [top study]
12. [bottom study]
13. [footwear study]

STRICT CONSISTENCY:
same exact face in every image
same exact hairstyle in every image
same exact outfit in every image
same exact colors in every image
same exact proportions in every image
same exact age and identity in every image
do not redesign the character between views

POSE AND PRESENTATION:
neutral standing pose
arms slightly away from body
orthographic turnaround feel
minimal perspective distortion
clean concept sheet presentation
production-ready character reference sheet
high detail
sharp facial features
clear clothing construction
clear material separation

COMPOSITION RULES:
arrange all parts in a compact grid

make the 4 full-body figures very tall and dominant

make the clothing and footwear studies useful and enlarged
