# Финальная самопроверка

Перед выдачей проверь текст внутренним голосом. Не выводи этот чек-лист пользователю.

Текст готов, только если:
- выбран подходящий режим по материалу;
- все факты взяты из `project.xlsx` и темы;
- первая фраза сразу вводит в материал;
- есть движение от хука к финальной мысли;
- каждое предложение связано с соседними;
- нет ощущения карточек, тезисов или нарезки;
- субъект действия понятен;
- имя человека при первом появлении имеет роль;
- нет ложного контраста;
- нет ленивых разнородных списков;
- нет канцелярита и машинной торжественности;
- нет запрещённых штампов;
- фразы удобно произносить вслух;
- объём соответствует лимиту;
- результат можно сразу отдавать диктору.

Если хотя бы один пункт нарушен, перепиши проблемное место и проверь снова.
