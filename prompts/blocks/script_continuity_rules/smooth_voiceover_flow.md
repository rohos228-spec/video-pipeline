# Связность и движение речи

Пиши для уха, а не для глаз. Слушатель слышит текст один раз, поэтому смысл должен быть понятен сразу.

Каждое предложение должно вести к следующему:
- причина переходит в действие;
- действие ведёт к следствию;
- следствие подготавливает новую часть;
- уточнение появляется только там, где без него нельзя двигаться дальше.

Не делай текст из коротких независимых предложений. Точка не должна заменять связь. Используй сложные предложения, союзы и мягкие переходы, если они помогают живой речи.

Ритм должен дышать:
- короткая фраза даёт паузу;
- средняя фраза ведёт основное движение;
- длинная фраза допустима, когда связывает причину, действие и результат.

Переходы должны быть почти незаметными. Не пиши «теперь перейдём», «далее рассмотрим», «также важно». Следующая мысль должна появляться из предыдущей.

Если кусок звучит как набор карточек, перепиши его как единый рассказ.
