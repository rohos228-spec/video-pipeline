# Long-form режим: цельный текст и ячейки

Используй этот блок только для длинного сценария, когда требуется текст около 10000–10500 символов и деление на ячейки.

Сначала напиши цельный рассказ целиком. Не думай о ячейках на первом проходе.

После финальной редакторской вычитки раздели текст на ячейки:
- каждая ячейка 110–140 символов с пробелами;
- каждая ячейка заканчивается на естественной речевой паузе;
- по возможности ячейка завершается целым предложением;
- соседние ячейки читаются слитно;
- граница ячейки не должна звучать как обрыв;
- ячейка не должна становиться отдельным мини-абзацем.

Если две соседние ячейки не склеиваются на слух, перепиши обе. Если строка закончилась только из-за лимита, а не из-за паузы, найди более естественную формулировку.

Деление на ячейки — техническая упаковка готового рассказа, а не способ писать текст.
