# Безопасные правки

Редактируй только то, что относится к текущей задаче. Любая правка должна быть локальной, объяснимой и обратимой.

Строгие правила:
- не переименовывай листы;
- не сдвигай строки и колонки;
- не удаляй существующие данные;
- не меняй формат таблицы без необходимости;
- не перезаписывай заполненные ячейки, если задача не требует заменить именно их;
- не создавай новые листы, если это не указано;
- не меняй id сцен, id кадров, id персонажей и id предметов без прямого основания;
- не заполняй соседние строки «заодно».

Если задача просит заполнить строку/диапазон, работай только с этим диапазоном. Если видишь конфликт между задачей и уже существующими данными, сохраняй структуру и выбирай минимальную правку.

Для массовых сцен двигайся по колонкам слева направо, чтобы не пропускать кадры и не смешивать данные разных сцен.
