# Вернуть файл

В ответ обязательно приложи **полный обновлённый `project.xlsx`**. Ответ без вложенного `.xlsx` не завершает шаг.

Формат результата:
- файл должен остаться обычным `.xlsx`, не `.zip`, не набором скриншотов и не текстовой таблицей;
- структура листов должна сохраниться;
- все старые данные, не относящиеся к задаче, должны остаться на месте;
- изменения должны быть внесены в нужный лист и нужный диапазон;
- после файла дай короткий текст: что изменено, какие строки/колонки затронуты, были ли пустые места из-за нехватки данных.

Не вставляй большой отчёт вместо файла. Файл — главный результат шага.
