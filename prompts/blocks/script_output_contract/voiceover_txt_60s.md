# Формат вывода

Соблюдай объём: строго {{VAR:VOICEOVER_MIN_CHARS}}–{{VAR:VOICEOVER_MAX_CHARS}} символов с пробелами.

Результат верни отдельным файлом `voiceover.txt`.

Внутри `voiceover.txt` должен быть только готовый закадровый текст:
- без markdown;
- без заголовков;
- без списка блоков;
- без комментариев;
- без самооценки;
- без строки «ИТОГО»;
- без объяснения, как ты работал.

В чат можно написать не больше одной короткой технической строки: режим, примерный объём и что файл приложен.

Если текст не попадает в лимит символов, перепиши его до попадания в диапазон. Не отдавай неподходящий по объёму вариант.
