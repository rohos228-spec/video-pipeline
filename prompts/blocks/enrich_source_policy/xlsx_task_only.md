# Только задача

Используй только данные из приложенного `project.xlsx`, текущей задачи и уже заданных промтов/настроек пайплайна.

Разрешено:
- брать смысл из строк текущего листа;
- учитывать соседние ячейки для понимания хронологии и контекста;
- использовать листы «персонажи», «предметы», «план» и другие существующие листы, если задача явно требует связать данные;
- оставлять ячейку пустой, если надёжной информации нет.

Запрещено:
- придумывать факты, персонажей, предметы, места и события;
- добавлять визуальные детали, которых нет в источнике, если задача требует фактологической точности;
- переносить данные из соседней сцены как новые факты текущей сцены;
- заполнять таблицу общими словами вместо конкретных наблюдаемых данных;
- исправлять весь проект, если задача просит один узкий слой.

Если информации недостаточно, лучше честно оставить пусто или кратко отметить ограничение в текстовом комментарии после файла. Не компенсируй пробелы выдумкой.
