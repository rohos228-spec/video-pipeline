# Проверка xlsx

Перед отправкой открой внутреннюю проверку результата как валидатор пайплайна.

Проверь:
- файл открывается как `.xlsx`;
- нужный лист существует и не переименован;
- шапка, строки и колонки не съехали;
- формулы/структура не повреждены;
- изменения внесены только в рамках задачи;
- в заполненных ячейках нет служебных комментариев, markdown-разметки или объяснений вместо данных;
- id персонажей и предметов соответствуют существующим листам, если они использовались;
- пустые ячейки оставлены пустыми осознанно, а не из-за пропуска;
- нет дублирования одной сцены в нескольких соседних колонках без причины.

Если после проверки находишь ошибку, исправь файл до отправки. Не отправляй файл с повреждённой структурой даже при частично выполненной задаче.
