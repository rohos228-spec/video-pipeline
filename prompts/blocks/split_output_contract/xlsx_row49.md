# Запись в xlsx

Запиши результат в `project.xlsx`, лист «план», строка 49 «закадровый текст».

Правило размещения:
- кадр 1 → `C49`;
- кадр 2 → `D49`;
- кадр 3 → `E49`;
- далее по колонкам слева направо до конца текста.

Не меняй название листа «план». Не редактируй другие строки без явной необходимости. Не удаляй существующую структуру таблицы.

Каждая заполненная ячейка должна содержать чистый текст озвучки без номера кадра, без кавычек-обёрток и без технических комментариев.

Верни полный обновлённый `.xlsx` файл. В чат добавь только короткую сводку: сколько блоков записано и в какие колонки.
