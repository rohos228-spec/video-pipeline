# Промт-проверка: ПРОМТЫ КАРТИНОК (шаг 6)

> ⚠️ ЭТО ЗАГЛУШКА. Замени её на реальный промт перед запуском Фазы 3 пайплайна.

Контракт ответа (читается парсером `app/services/gpt_check.py`):

* Если excel-промтов одобрен — ответь свободным текстом, содержащим
  слово **«одобрено»** (или `approved`).
* Если есть правки — отправь ОБНОВЛЁННЫЙ excel файлом (📎). Парсер
  заменит старый.
* В крайнем случае — ответь фразой, начинающейся со слова
  **«перегенерация»**, и инструкцией.
