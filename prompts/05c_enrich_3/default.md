# Доп работа с EXCEL #3 — заполнение листа «Персонажи»

Открой приложенный `project.xlsx`. Работай **только** с листом **«Персонажи»** и строкой **49** листа **«план»** (текст сцены / закадрового блока для главного героя).

## Задача

1. Прочитай строку **49** листа «план» (столбец B = первая сцена, далее по столбцам).
2. Определи **одного** главного персонажа ролика по смыслу этого текста (и соседних сцен при необходимости).
3. Заполни **столбец B** на листе «Персонажи» для id **c01** (главный герой).

## Формат листа «Персонажи» (обязательно)

Колонка **A** — подписи строк, данные персонажа — **по столбцам** (B, C, …):

| Строка | Поле |
|--------|------|
| **1** | id персонажа → `c01` |
| **3** | Имя |
| **4** | Внешность (возраст, пол, телосложение, лицо, волосы, кожа — **как в сценарии**) |
| **5** | Одежда |
| **6** | Характер / роль |
| **7** | Правила (референсы на других id, если нужно; иначе пусто) |

**Не путай номера строк:** id `c01` пишется в **строку 1**, не в строку 2.

## Вид персонажа (критично)

- Если в плане / сценарии герой **человек** — описывай **человека**. Не превращай его в кота, зверя или антропоморфное животное.
- Если в тексте явно другой вид (кот, робот и т.д.) — сохраняй **тот же** вид.
- **Не** подставляй правила из других шагов пайплайна («все персонажи — коты», «антропоморфные коты»), если в **этом** ролике герой человек.
- Не копируй внешность из промтов кадров на листе «план» (строки 50–52), если они противоречат строке 49.

## Качество

- Пиши по-русски, конкретно, без воды.
- Не выдумывай лишних персонажей в других столбцах, если их не просили.
- Не меняй другие листы и столбцы, кроме необходимого заполнения **c01** в «Персонажи».

## Ответ

Сохрани все изменения в xlsx и **обязательно приложи обновлённый `project.xlsx` файлом** в ответ. Кратко текстом: что заполнил для c01.
