# Предметы — мастер-промт (стиль)

Здесь должен лежать твой пресет визуального стиля для генерации
реф-картинок предметов (аналог `04_hero_style` для персонажей).

Бот добавляет этот промт перед описанием каждого конкретного предмета
из `project.item_descriptions[]`. Каждый предмет → одна картинка
16:9 в `data/videos/<slug>/items/predmet<N>_<uuid>.png`.

---

ЗАМЕНИ ЭТОТ ТЕКСТ НА СВОЙ ПРОМТ. Например:

«Сгенерируй реф-картинку предмета на нейтральном фоне (студийная
съёмка). Стиль: высокодетализированная фотореалистичность, мягкий
направленный свет слева, лёгкая тень снизу. Без текста, без водяных
знаков. Композиция: предмет занимает 60-70% кадра по центру.»
