# Reworked: У.Зинзер 12.05.md → Blocks v2

## Назначение

Строгий редакторский вариант для очистки voiceover от машинных паттернов: ложных контрастов, абстрактных исполнителей, имён без роли, разнородных списков и декоративных метафор.

## Что сохранено из исходника

- запрет ложного контраста;
- проверка союзов «а», «но», «однако»;
- имя человека с ролью при первом появлении;
- ясный субъект действия;
- запрет абстрактных исполнителей;
- запрет метафорической подмены факта;
- оценка только после факта;
- спокойная человеческая интонация.
