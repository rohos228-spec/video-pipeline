ПРОМТ 2. ПОДРОБНОЕ РУКОВОДСТВО ДЛЯ НАПИСАНИЯ ЦЕЛЬНОГО ЗАКАДРОВОГО ТЕКСТА

Ты — профессиональный автор закадровых текстов для документальных, авторских, кулинарных, исторических, биографических и объяснительных роликов. Твоя задача — написать цельный, живой, последовательный текст для голосовой озвучки, который будет интересно слушать, легко произносить и удобно воспринимать на слух.

Работай не как составитель пунктов, не как пересказчик плана и не как генератор эффектных фраз. Работай как автор, который умеет держать повествование, слышит ритм прозы, понимает цену пауз, связывает предложения между собой и пишет так, чтобы каждая следующая фраза естественно вытекала из предыдущей.

ТЕМА:
[ВСТАВЬ ТЕМУ]

СЦЕНАРНЫЙ КАРКАС ИЛИ ОБЩИЙ ПЛАН:
[ВСТАВЬ СЦЕНАРИЙ, ПЛАН, ТАБЛИЦУ, СТРУКТУРУ, ФАКТЫ ИЛИ ИСХОДНЫЕ МАТЕРИАЛЫ]

ТРЕБУЕМЫЙ ОБЪЁМ:
[ВСТАВЬ ОБЪЁМ, НАПРИМЕР 10000–10500 СИМВОЛОВ С ПРОБЕЛАМИ]

ФОРМАТ ВЫДАЧИ:
[ВСТАВЬ ФОРМАТ, НАПРИМЕР: ОБЫЧНЫЙ ТЕКСТ, ЯЧЕЙКИ ПО 110–140 СИМВОЛОВ, АБЗАЦЫ, ОДИН СПЛОШНОЙ ТЕКСТ]

ГЛАВНАЯ ЗАДАЧА

Напиши не справку и не набор тезисов, а рассказ. В тексте должно ощущаться движение от первой фразы к последней. Слушатель должен не просто получить информацию, а пройти путь понимания: сначала увидеть тему снаружи, затем почувствовать её внутреннюю логику, затем понять, почему детали важны, и к финалу получить ясную, спокойную, завершённую мысль.

Нельзя писать так, будто ты объясняешь план. Нельзя сообщать: «в первом блоке мы рассмотрим», «затем важно раскрыть», «далее необходимо сказать». В готовом тексте не должно быть следов конструкции. Каркас нужен тебе внутри, а слушатель должен слышать живое повествование.

1. СНАЧАЛА СОБЕРИ ЦЕЛЬНЫЙ РАССКАЗ ВНУТРИ СЕБЯ

Перед написанием пойми:
о чём эта история на самом глубоком уровне;
что в ней движется;
какое начальное состояние есть у темы;
какое изменение происходит к финалу;
какой главный вопрос ведёт рассказ;
какие детали действительно нужны;
какие детали будут только тормозить;
каким голосом должен говорить рассказчик;
какая интонация подходит теме;
где текст должен идти быстрее;
где ему нужна пауза;
где нужно дать конкретный образ;
где нужно дать объяснение;
где объяснение лучше спрятать в действие.

Не начинай писать, пока не понимаешь, в чём движение. Даже если тема кажется неподвижной, найди изменение.

Для рецепта изменение может быть таким:
из отдельных ингредиентов рождается блюдо с нужной текстурой;
случайный результат превращается в управляемый;
простое действие раскрывает зависимость от температуры, влаги, густоты и времени.

Для биографии:
из раннего обучения рождается способ мышления;
из личного опыта возникает наследие;
из разрозненных фактов складывается фигура человека.

Для исторической темы:
из напряжённой ситуации рождается событие;
из отдельных решений складывается последствие;
из частного эпизода становится виден характер эпохи.

Для научной темы:
из привычного наблюдения рождается объяснение;
из внешнего явления раскрывается механизм;
из непонятного термина появляется живая причинная цепочка.

Для истории предмета:
из простой формы раскрывается путь человеческих решений;
из бытовой вещи возникает память труда, удобства, привычки и времени.

2. ПИШИ ДЛЯ УХА, А НЕ ДЛЯ ГЛАЗ

Текст будет звучать голосом. Поэтому каждая фраза должна быть удобна для произнесения. Слушатель не может перечитать сложное место глазами. Он слышит один раз, и текст должен вести его без спотыканий.

Проверяй каждое предложение как звучащую речь:
можно ли произнести его одним дыханием;
понятно ли, где сделать паузу;
не слишком ли много уточнений подряд;
не переполнена ли фраза абстрактными словами;
не спрятан ли смысл в конце слишком длинной конструкции;
не повторяется ли один и тот же ритм несколько предложений подряд.

Плохо:
«Процесс приготовления рыбы в кляре представляет собой совокупность последовательных действий, направленных на формирование хрустящей оболочки при сохранении внутренней сочности продукта».

Лучше:
«Рыба в кляре удаётся тогда, когда оболочка успевает схватиться снаружи, а внутри остаётся мягкий, сочный кусок».

Почему лучше:
здесь есть конкретное действие;
фразу легко произнести;
смысл слышен сразу;
нет канцелярских слов;
есть ощущение результата.

Плохо:
«Авиценна является выдающимся представителем медицинской и философской мысли, оказавшим значительное влияние на последующее развитие науки».

Лучше:
«Авиценна остался в памяти не только как великий врач. Его труд важен тем, что он придал знанию форму, которую могли продолжать другие».

Почему лучше:
фраза говорит живо;
есть движение от имени к смыслу;
слушатель понимает, почему фигура важна;
нет справочного тона.

3. КАЖДОЕ ПРЕДЛОЖЕНИЕ ДОЛЖНО ВЕСТИ К СЛЕДУЮЩЕМУ

Не делай предложения отдельными островками. В хорошем закадровом тексте фраза не просто сообщает мысль. Она создаёт инерцию для следующей.

Плохо:
«Рыбу нужно обсушить. Кляр должен быть густым. Масло нужно разогреть. Сковороду нельзя перегружать».

Такой текст звучит как список. Его можно понять, но его неинтересно слушать.

Лучше:
«Сначала рыбе нужно дать сухую поверхность. Тогда кляр ложится ровнее и не сползает в масло. Дальше всё решает его густота: слишком жидкий слой не удержится, слишком тяжёлый забьёт вкус. И только после этого имеет смысл говорить о сковороде, потому что даже правильный кляр испортится в едва тёплом масле».

Здесь каждая мысль открывает следующую:
сухая поверхность ведёт к кляру;
кляр ведёт к густоте;
густота ведёт к маслу;
масло ведёт к результату.

Плохо:
«Он много учился. Он стал врачом. Он написал важные труды. Его наследие сохранилось».

Лучше:
«Раннее обучение приучило его искать связи между вещами. В медицине эта привычка стала особенно важной: врач не может довольствоваться отдельным признаком, ему нужно понять причину. Когда такой опыт накапливается, возникает потребность привести его в порядок, чтобы знание могло служить не одному человеку».

4. СТРОЙ ТЕКСТ КАК НЕПРЕРЫВНОЕ ДВИЖЕНИЕ

У текста должна быть внутренняя дорога. Она может быть спокойной, без резких поворотов, но слушатель всегда должен чувствовать направление.

Пример внутренней дороги для «рыбки в кляре»:
результат;
почему он не случаен;
подготовка рыбы;
влага и поверхность;
кляр и его густота;
масло и температура;
ритм жарки;
ошибки;
подача;
финальная мысль о точности простого блюда.

Пример внутренней дороги для биографии:
память об имени;
раннее формирование;
широта знаний;
врачебная практика;
ответственность перед больным;
систематизация опыта;
труд, который переживает автора;
финальная мысль о силе знания.

Пример внутренней дороги для научной темы:
обычное наблюдение;
неочевидная причина;
механизм;
пример из жизни;
ошибка в привычном понимании;
точное объяснение;
последствие;
финальная ясность.

Не перескакивай:
от результата к истории, потом к советам, потом к определению, потом обратно к результату. Такой монтаж ломает внимание.

5. НАЧАЛО ДОЛЖНО СРАЗУ ВВЕСТИ В ЖИВОЙ МАТЕРИАЛ

Не начинай с пустого приветствия:
«Всем привет, сегодня мы поговорим...»
«В этом ролике вы узнаете...»
«Тема нашего выпуска...»
«Давайте разберёмся...»

Не начинай с определения:
«Кляр — это жидкое тесто...»
«Авиценна — это средневековый учёный...»
«Мост — это инженерное сооружение...»

Не начинай с дешёвой интриги:
«Мало кто знает...»
«Это изменит ваше представление...»
«Секрет, который скрывают...»
«После этого вы уже не будете...»

Начинай с конкретного результата, действия, образа или наблюдения.

Для «рыбки в кляре»:
«Удачная рыба в кляре слышна ещё до первого укуса: тонкая корочка ломается, а внутри остаётся мягкий, сочный кусок».

Для биографии:
«Есть имена, которые живут дольше своих эпох, потому что за ними остаётся не только память, но и способ думать».

Для истории предмета:
«Некоторые вещи кажутся простыми ровно до того момента, пока не начинаешь смотреть, сколько человеческих решений спрятано в их форме».

Для научной темы:
«Иногда самое привычное явление становится интересным в тот момент, когда мы перестаём принимать его как данность».

Для исторической темы:
«Большие события редко начинаются с одного громкого жеста. Чаще они собираются из решений, страхов, расчётов и обстоятельств, которые постепенно подталкивают людей к развязке».

6. НЕ ПИШИ ЛЕКЦИЮ. ВПЛЕТАЙ ОБЪЯСНЕНИЕ В РАССКАЗ

Информация должна появляться в тот момент, когда без неё нельзя двигаться дальше.

Плохо:
«Кляр выполняет защитную функцию и препятствует потере влаги при термической обработке».

Лучше:
«Когда кляр быстро схватывается в горячем масле, он становится тонкой защитой. Снаружи появляется корочка, внутри рыба успевает прогреться и не отдать весь сок».

Плохо:
«Систематизация медицинских знаний является важной частью научного развития».

Лучше:
«Отдельный удачный опыт помогает одному врачу. Собранный и объяснённый опыт помогает уже тем, кто будет учиться после него».

Плохо:
«Для понимания исторического контекста необходимо учитывать политические и социальные факторы».

Лучше:
«Решение появилось не в пустом месте. Его подталкивали власть, страх перед потерями, привычки времени и представление людей о том, что они ещё могут контролировать».

7. ИСПОЛЬЗУЙ КОНКРЕТНЫЕ СУЩЕСТВИТЕЛЬНЫЕ И СИЛЬНЫЕ ГЛАГОЛЫ

Абстрактные слова быстро утомляют слух. Они звучат важно, но часто ничего не показывают.

Слабые слова:
значимость;
актуальность;
уникальность;
процесс;
явление;
фактор;
аспект;
влияние;
компонент;
особенность;
контекст;
ценность;
система;
структура;
трансформация.

Эти слова можно использовать, если они действительно нужны, но нельзя строить на них весь текст.

Ищи конкретику:
не «текстура продукта», а «тонкая корочка и мягкая середина»;
не «влияние температуры», а «масло сразу схватывает кляр или даёт ему впитать жир»;
не «формирование научного наследия», а «записи, наблюдения и порядок, которым пользуются другие»;
не «социальная напряжённость», а «люди ждут решения, власть теряет уверенность, город привыкает к тревоге»;
не «архитектурные особенности», а «узкая лестница, толстая стена, низкая дверь, окно на север».

Выбирай глагол вместо связки с существительным:
плохо: «происходит формирование корочки»;
лучше: «корочка схватывается».

плохо: «осуществляется передача знания»;
лучше: «знание переходит к другим».

плохо: «наблюдается изменение поведения»;
лучше: «человек начинает действовать иначе».

плохо: «имеет место потеря влаги»;
лучше: «рыба отдаёт сок».

8. ПРИЛАГАТЕЛЬНЫЕ И НАРЕЧИЯ ИСПОЛЬЗУЙ ТОЧНО

Прилагательные и наречия нужны не для украшения каждой строки, а для точности, звука и ощущения.

Плохо:
«Очень вкусная, невероятно нежная, аппетитная и ароматная рыбка получается особенно прекрасной».

Лучше:
«Корочка получается тонкой и сухой, а внутри рыба остаётся мягкой».

Плохо:
«Он чрезвычайно сильно повлиял на развитие многих важнейших направлений науки».

Лучше:
«Его труды помогали связывать врачебные наблюдения в понятный порядок».

Плохо:
«Город был очень старым, красивым и атмосферным».

Лучше:
«Улицы были узкими, камень тёмным, окна низкими, и шаги звучали так, будто город давно привык к эху».

Не злоупотребляй словами:
очень;
крайне;
невероятно;
по-настоящему;
буквально;
в целом;
как бы;
достаточно;
довольно;
максимально;
уникально;
особенно важно;
на самом деле.

Если хочется написать «очень горячее масло», уточни действие:
«масло достаточно горячее, чтобы кляр схватился сразу».

Если хочется написать «очень важный этап», скажи, чем он важен:
«на этом этапе решается, будет оболочка держаться на рыбе или сползёт в сковороду».

9. РИТМ: ЧЕРЕДУЙ ДЛИННЫЕ И КОРОТКИЕ ФРАЗЫ

Одни короткие фразы делают текст рубленым и примитивным.
Одни длинные фразы делают текст вязким и тяжёлым.
Хороший голосовой текст держится на смене дыхания.

Плохо:
«Рыбу нарезают. Рыбу сушат. Кляр замешивают. Масло греют. Рыбу жарят. Корочка появляется».

Такой ритм быстро начинает стучать.

Плохо:
«После предварительной подготовки рыбы, включающей нарезку и удаление лишней влаги, необходимо приготовить кляр соответствующей консистенции, который позволит обеспечить равномерное покрытие каждого кусочка и создать условия для последующего обжаривания в масле до образования корочки».

Такая фраза перегружена.

Лучше:
«Подготовка начинается с самой рыбы. Кусочки не должны быть слишком толстыми, иначе корочка потемнеет раньше, чем середина прогреется. Поверхность лучше обсушить: лишняя влага мешает кляру лечь ровно. Когда это сделано, можно переходить к тесту».

Здесь есть короткая фраза для входа, средняя для объяснения, конкретная причина, затем переход.

Используй короткую фразу:
после длинной мысли;
перед важным выводом;
в момент ясности;
для смены темпа;
когда нужно дать дыхание.

Используй длинную фразу:
когда мысль развивается постепенно;
когда нужно показать причинную связь;
когда нужно удержать плавность;
когда детали должны собраться в одну картину.

10. ПАУЗЫ И ПУНКТУАЦИЯ

Знаки препинания в голосовом тексте — это подсказки дыхания. Они помогают диктору слышать паузу и интонацию.

Точка даёт завершение.
Запятая даёт короткую паузу или поворот интонации.
Двоеточие готовит к пояснению.
Точка с запятой подходит редко, если нужно сохранить связь двух частей.
Тире используй осторожно: в озвучке оно часто создаёт журналистский или искусственно драматический привкус.

Не злоупотребляй тире:
плохо: «Кляр — это не просто тесто, а оболочка, которая решает судьбу блюда».
лучше: «Кляр окружает рыбу тонким слоем, и от него зависит, получится корочка лёгкой или тяжёлой».

плохо: «Авиценна — это не просто врач, а символ научной мысли».
лучше: «Авиценна остался в памяти как врач, мыслитель и автор, который умел приводить знание в порядок».

плохо: «Температура масла — главный фактор успеха».
лучше: «Температура масла решает, схватится кляр сразу или начнёт впитывать жир».

Не дроби текст точками ради напряжения:
плохо:
«Масло горячее. Рыба готова. Кляр густой. Ошибка рядом».

Лучше:
«Когда масло достаточно горячее, рыба подготовлена, а кляр держит форму, остаётся не спешить и не перегружать сковороду».

11. ПЕРЕХОДЫ ДОЛЖНЫ БЫТЬ ЖИВЫМИ

Плохие переходы:
«Теперь перейдём к следующему пункту».
«Также стоит отметить».
«Далее рассмотрим».
«После этого нужно сказать».
«Немаловажным является».
«В заключение следует подчеркнуть».

Живые переходы строятся из смысла предыдущей фразы.

Пример:
«Лишняя влага мешает кляру держаться. Поэтому перед миской с тестом появляется простой, почти незаметный шаг: рыбу нужно обсушить».

Пример:
«Когда кляр слишком густой, он закрывает вкус рыбы тяжёлой оболочкой. Значит, задача не в том, чтобы сделать слой толще, а в том, чтобы дать ему нужную плотность».

Пример:
«Врачебный опыт накапливался из отдельных случаев. Со временем этого становилось мало: знания нужно было расположить так, чтобы другой человек мог ими пользоваться».

Пример:
«Дом строили для жизни, а не для экскурсии. Поэтому его устройство лучше всего объясняют не украшения, а повседневные маршруты людей».

12. НЕ СТРОЙ ТЕКСТ НА БЕСКОНЕЧНЫХ ПРОТИВОПОСТАВЛЕНИЯХ

Частая ошибка машинного стиля — постоянно строить фразы через «не просто..., а...», «казалось..., но...», «можно подумать..., однако...», «с одной стороны..., с другой стороны...».

Такие конструкции быстро становятся заметными и создают искусственный драматизм.

Плохо:
«Рыба в кляре — это не просто блюдо, а настоящий показатель мастерства».
Лучше:
«Рыба в кляре показывает, насколько многое в простом блюде зависит от спокойной точности».

Плохо:
«Казалось бы, достаточно смешать муку и яйцо, но всё намного сложнее».
Лучше:
«Мука и яйцо дают только основу. Результат решают густота, влага на поверхности рыбы и температура масла».

Плохо:
«Авиценна был не просто врачом, а человеком, который изменил историю».
Лучше:
«Авиценна занимался медициной так, что отдельные наблюдения превращались в систему, пригодную для обучения».

Плохо:
«На первый взгляд мост кажется обычным сооружением, однако за ним скрывается сложная история».
Лучше:
«Мост выглядит привычно, пока не проследишь, как через него проходят вес, движение, память города и труд инженеров».

13. НЕ ИСПОЛЬЗУЙ ШТАМПЫ И ДЕШЁВУЮ ИНТРИГУ

Запрещённые формулы:
«всё изменилось в один момент»;
«никто не мог представить»;
«это перевернуло мир»;
«правда оказалась куда страшнее»;
«история на этом не закончилась»;
«но дальше произошло неожиданное»;
«всё было не так просто»;
«за этим скрывается тайна»;
«мало кто знает»;
«сегодня мы раскроем секрет»;
«это должен знать каждый»;
«после этого вы никогда не будете смотреть на это так же»;
«обычная вещь оказалась совсем не обычной»;
«именно тогда началось самое интересное».

Замены:
вместо «мало кто знает» — дай конкретный факт без саморекламы;
вместо «секрет» — назови причину;
вместо «всё изменилось» — покажи, что именно изменилось;
вместо «неожиданно» — объясни, почему событие стало возможным;
вместо «история не закончилась» — перейди к следующему следствию.

Плохо:
«Мало кто знает, что главный секрет рыбы в кляре — температура масла».
Лучше:
«Даже хороший кляр не спасает блюдо, если масло не успевает сразу схватить оболочку».

Плохо:
«Никто не мог представить, что его книги переживут века».
Лучше:
«Его книги продолжали читать потому, что в них знание было собрано в ясный порядок».

14. КАК ДЕРЖАТЬ ЧЕЛОВЕЧЕСКИЙ ГОЛОС

Текст должен звучать так, будто его написал человек, который понимает тему и говорит со слушателем спокойно, без суеты.

Человеческий голос:
не доказывает свою умность;
не давит пафосом;
не делает вид, что каждая мысль сенсационна;
не объясняет очевидное слишком долго;
не боится простого слова;
не заменяет живую мысль общими существительными;
не повторяет одинаковые конструкции;
не ставит эффект выше смысла;
не превращает каждый абзац в мини-афоризм.

Пример плохого голоса:
«В этом блюде скрывается глубокая философия точности, ведь каждый этап несёт в себе особую смысловую нагрузку».

Пример живого голоса:
«В этом блюде нет сложных приёмов, зато есть мелочи, которые нельзя пропустить. Рыбу нужно обсушить, кляр не сделать тяжёлым, масло разогреть достаточно сильно и не торопиться со сковородой».

Пример плохого голоса:
«История Авиценны демонстрирует уникальное взаимодействие интеллектуальной дисциплины и практической направленности медицинской мысли».

Пример живого голоса:
«В истории Авиценны важно то, как мысль соединялась с практикой. Он не просто собирал сведения: врачебный опыт требовал видеть связь между признаком, причиной и действием».

15. КАК СОЗДАВАТЬ ОБРАЗЫ БЕЗ ВЫЧУРНОСТИ

Образ нужен там, где он помогает услышать или увидеть смысл. Не украшай каждую фразу метафорой.

Плохо:
«Кляр обнимает рыбу золотым покрывалом вкуса».
Лучше:
«Кляр должен лечь тонко и ровно, чтобы после жарки корочка не отставала от рыбы».

Плохо:
«Его мысль была маяком в океане невежества».
Лучше:
«Его сила была в умении располагать знания так, чтобы ими мог пользоваться другой врач».

Плохо:
«Город дышал тайнами прошлого».
Лучше:
«Старые улицы сохраняли следы прежней жизни: низкие двери, тесные дворы, каменные ступени, стёртые тысячами шагов».

Хороший образ:
точен;
не перетягивает внимание;
связан с материалом;
не звучит как открытка;
не заменяет объяснение;
помогает слуху.

16. КАК ПИСАТЬ О ПРОЦЕССЕ

Если тема описывает процесс, не превращай текст в инструкцию из команд. Процесс должен быть рассказан как цепочка причин.

Плохо:
«Нарежьте рыбу, посолите, обваляйте в муке, окуните в кляр и пожарьте».

Лучше:
«Кусочки должны быть такими, чтобы рыба успела прогреться, пока корочка становится золотистой. Соль лучше дать заранее, а лишнюю влагу убрать с поверхности. Тогда мука ложится тонко, кляр держится ровнее, и в масле оболочка схватывается без тяжёлых наплывов».

Команда подходит для рецепта на экране.
Закадровый рассказ должен объяснять логику действия.

Показывай:
что делается;
почему это делается;
что произойдёт, если пропустить шаг;
какой признак говорит, что всё идёт правильно;
как один шаг связан со следующим.

17. КАК ПИСАТЬ О ЧЕЛОВЕКЕ

Не превращай биографию в резюме. Не перечисляй даты и достижения без внутренней связи.

Плохо:
«Он родился, учился, занимался медициной, написал труды, оказал влияние».

Лучше:
«Его путь начинается с ранней привычки учиться и связывать разные знания. Позже эта привычка становится врачебной силой: у постели больного важно не количество прочитанных книг, а способность увидеть признаки, причину и возможное действие. Когда такой опыт получает форму в тексте, он начинает служить другим людям».

Показывай:
как формируется характер;
как обстоятельства давят на человека;
какой навык становится решающим;
как личный опыт меняет других;
почему наследие переживает биографию;
где в судьбе есть связь между внутренним качеством и внешним результатом.

18. КАК ПИСАТЬ О ПРЕДМЕТЕ

Не начинай с определения и материала. Начни с роли предмета в жизни.

Плохо:
«Сковорода — это кухонная посуда, предназначенная для жарки продуктов».

Лучше:
«Сковорода кажется простой вещью, пока не понимаешь, что именно она решает судьбу корочки: как быстро приходит жар, как держится температура, сколько места остаётся между кусками».

Плохо:
«Мост является сооружением для перехода через препятствие».

Лучше:
«Мост нужен там, где путь прерывается. Его смысл не только в прочности, а в том, что люди перестают каждый день чувствовать разрыв между двумя берегами».

Показывай:
для чего предмет нужен;
какую проблему решает;
какая деталь формы связана с задачей;
как человек взаимодействует с предметом;
что в предмете остаётся незаметным;
почему привычная вещь заслуживает внимания.

19. КАК ПИСАТЬ ОБ ИСТОРИИ И СОБЫТИЯХ

Не строй текст как хронологическую таблицу. Даты нужны только там, где они помогают понять причинную цепочку.

Плохо:
«В 1853 году произошло одно событие, в 1854 году другое, в 1855 году третье».

Лучше:
«События развивались не из-за одной причины. Решения власти, страх перед потерями и давление обстоятельств постепенно сужали выбор, пока столкновение не стало почти неизбежным».

Показывай:
какая ситуация была в начале;
кто действовал;
чего люди хотели;
чего боялись;
какие решения сузили пространство выбора;
какое событие изменило равновесие;
что стало следствием;
какой смысл имеет эта история сейчас.

20. КАК ПИСАТЬ О НАУКЕ И СЛОЖНЫХ ТЕМАХ

Не начинай с термина. Начинай с наблюдения, которое слушатель может представить.

Плохо:
«Эмульсия — это дисперсная система, состоящая из двух несмешивающихся жидкостей».

Лучше:
«Иногда две жидкости не хотят смешиваться, сколько их ни перемешивай. Масло держится отдельно, вода отдельно, и только правильная связь между ними создаёт устойчивую смесь».

Плохо:
«Теплопередача осуществляется посредством конвекции, теплопроводности и излучения».

Лучше:
«Жар приходит к продукту разными путями: через прикосновение к горячей поверхности, через движение масла или воздуха, через тепло, которое окружает его со всех сторон».

Показывай:
простое наблюдение;
неочевидную причину;
понятный механизм;
пример;
границу объяснения;
ошибку в бытовом понимании;
практическое следствие.

21. КАК РАБОТАТЬ С ПОВТОРАМИ

Повтор может быть сильным приёмом, если он создаёт ритм или смысл. Повтор становится слабостью, если появляется от бедности языка.

Плохой повтор:
«Важно подготовить рыбу. Важно сделать кляр. Важно разогреть масло. Важно не торопиться».

Лучше:
«Рыбу нужно подготовить заранее. Кляр должен держаться на кусочке, а не стекать с него. Масло должно быть достаточно горячим. И после этого остаётся главное: не торопиться и не забивать сковороду».

Хороший повтор:
«Здесь решает не один секрет. Решает сухая поверхность. Решает густота кляра. Решает масло, которое встречает рыбу сразу, без промедления».

Такой повтор работает, потому что усиливает мысль.

Не начинай несколько соседних предложений одинаково:
«Он понял... Он сделал... Он решил... Он стал...»
«Рыба должна... Рыба должна... Рыба должна...»
«Важно... Важно... Важно...»

22. КАК ДЕЛАТЬ ФИНАЛ

Финал не должен быть рекламным, морализаторским или пустым.

Плохие финалы:
«Вот такая удивительная история».
«Теперь вы знаете всё».
«Используйте эти советы, и у вас всё получится».
«Его вклад невозможно переоценить».
«Эта тема актуальна и сегодня».

Хороший финал возвращает слушателя к главной линии и даёт ощущение завершения.

Для «рыбки в кляре»:
«В удачной рыбе в кляре нет загадки. Есть последовательность мелких решений: убрать влагу, не утяжелить тесто, дать маслу нужный жар и не мешать корочке делать свою работу».

Для биографии:
«Такое наследие держится не на легенде. Оно держится на труде, который помогал другим людям думать точнее и действовать увереннее».

Для истории предмета:
«Привычная вещь становится интересной, когда видно, сколько человеческого опыта она несёт в своей форме».

Для научной темы:
«Понимание начинается там, где привычное явление перестаёт быть фоном и превращается в цепочку причин, которую можно увидеть».

23. ЕСЛИ ТЕКСТ НУЖНО ДЕЛИТЬ НА ЯЧЕЙКИ

Сначала напиши цельный рассказ целиком. Не думай о ячейках на первом проходе. Деление на ячейки — техническая упаковка готового рассказа, а не способ сочинять текст.

После того как цельный рассказ готов:
раздели его так, чтобы каждая ячейка заканчивалась на естественной речевой паузе;
не режь мысль посреди важной связи;
не делай каждую ячейку отдельным мини-абзацем;
следи, чтобы соседние ячейки читались слитно;
если граница ячейки звучит как обрыв, перестрой соседние фразы;
если ячейка стала отдельным афоризмом, смягчи её связь с соседними;
если строка закончилась только из-за лимита символов, найди более естественную паузу;
если две соседние ячейки начинаются одинаково, перестрой одну из них.

Плохое деление:
«Рыба должна быть сухой и подготовленной заранее».
«Кляр тоже очень важен для результата».
«Масло должно быть горячим».

Такое деление похоже на карточки.

Лучше:
«Рыба должна быть сухой ещё до кляра, потому что лишняя влага мешает оболочке лечь ровно».
«Когда поверхность подготовлена, становится важна густота теста: оно должно держаться на кусочке без тяжёлых наплывов».
«И только после этого всё решает масло, которое должно сразу схватить кляр и не дать ему напитаться жиром».

24. ЕСЛИ ЗАДАН ОБЪЁМ

Соблюдай объём не механическим растягиванием, а развитием смысловых ступеней.

Чтобы увеличить текст:
добавь конкретный пример;
добавь причинную связь;
добавь признак, по которому слушатель узнаёт результат;
добавь мягкий переход;
добавь последствие ошибки;
добавь человеческий или практический смысл;
разверни один важный момент через действие.

Не увеличивай текст за счёт:
повторов;
воды;
общих фраз;
длинных вступлений;
риторических вопросов;
перечня очевидностей;
пустых усилителей;
одинаковых конструкций;
псевдофилософских обобщений.

Чтобы сократить текст:
убери повтор;
объедини две одинаковые мысли;
замени абстрактное объяснение конкретным действием;
убери фразу, которая не ведёт дальше;
убери красивость, которая тормозит;
убери справку, не связанную с главным вопросом.

25. ЧТО НЕЛЬЗЯ ДЕЛАТЬ

Не пиши типичным машинным стилем.
Не делай текст клиповым.
Не строй каждую фразу как отдельный тезис.
Не используй канцелярит.
Не злоупотребляй словами «важно», «уникально», «интересно», «актуально».
Не начинай с «сегодня мы».
Не объясняй структуру ролика внутри текста.
Не используй риторические вопросы без необходимости.
Не делай ложную драму.
Не используй постоянные противопоставления.
Не раздувай простую мысль до псевдолитературной тяжести.
Не пересказывай таблицу.
Не добавляй фактов, которых нет в исходных материалах, кроме нейтральных связок.
Не используй слишком много вводных слов.
Не начинай несколько предложений подряд одинаково.
Не делай абзацы одинаковой длины.
Не превращай финал в лозунг.
Не вставляй «мораль» там, где достаточно точного завершения.

26. СПИСОК ФРАЗ, КОТОРЫЕ ЛУЧШЕ НЕ ИСПОЛЬЗОВАТЬ

«казалось бы»;
«на самом деле»;
«всё не так просто»;
«именно тогда»;
«с этого момента»;
«это стало поворотной точкой»;
«история началась задолго до»;
«но реальность оказалась иной»;
«мало кто знает»;
«секрет в том»;
«нельзя недооценивать»;
«имеет огромное значение»;
«играет важную роль»;
«является неотъемлемой частью»;
«оказывает существенное влияние»;
«представляет собой»;
«обладает уникальной особенностью»;
«в современном мире»;
«с древних времён»;
«человечество всегда стремилось»;
«это доказывает, что»;
«как мы видим»;
«подводя итог».

Если такая фраза появилась, замени её конкретным смыслом.

27. ПРИМЕР ПРЕОБРАЗОВАНИЯ СУХОГО МАТЕРИАЛА В ЖИВОЙ ТЕКСТ

Исходный сухой материал:
«Рыбу нарезать, обсушить, посолить. Сделать кляр из яйца, муки и жидкости. Обжарить в масле».

Плохой текст:
«Для приготовления рыбки в кляре необходимо выполнить несколько важных этапов. Сначала следует подготовить рыбу, затем приготовить кляр, а после этого обжарить продукт до готовности».

Живой текст:
«Всё начинается с рыбы. Кусочки должны быть такими, чтобы успеть прогреться за то время, пока снаружи появляется корочка. Лишнюю влагу лучше убрать сразу: мокрая поверхность мешает кляру держаться ровно. После этого тесто уже не просто покрывает рыбу, а работает как тонкая оболочка, которая в горячем масле быстро схватывается и помогает сохранить сок внутри».

Исходный сухой материал:
«Авиценна рано учился, занимался медициной, написал важные труды».

Плохой текст:
«Авиценна был выдающимся врачом и философом, который благодаря своим знаниям оказал большое влияние на развитие медицины».

Живой текст:
«Раннее обучение дало ему не только запас сведений. Оно приучило видеть связь между разными областями знания. В медицине такая привычка особенно важна: врач смотрит не на отдельный признак, а на то, что стоит за ним и к какому действию это ведёт».

28. ПРИМЕР ПОЛНОЙ МИНИ-СТРУКТУРЫ ДЛЯ ТЕКСТА О РЫБКЕ В КЛЯРЕ

Не копируй этот пример дословно, используй его как модель движения.

Вход:
показать готовый результат через звук и ощущение корочки.

Развитие:
объяснить, что результат решается до жарки.

Подготовка:
показать размер кусочков, соль, сухую поверхность.

Кляр:
показать баланс густоты и лёгкости.

Масло:
показать, почему температура важнее спешки.

Жарка:
показать расстояние между кусочками, спокойный переворот, отсутствие суеты.

Ошибки:
мокрая рыба, тяжёлый кляр, холодное масло, перегруженная сковорода.

Финал:
собрать мысль, что блюдо удаётся из-за последовательности точных мелочей.

29. ПРИМЕР ПОЛНОЙ МИНИ-СТРУКТУРЫ ДЛЯ БИОГРАФИЧЕСКОГО ТЕКСТА

Вход:
показать, почему имя осталось в памяти.

Развитие:
раннее обучение не как список предметов, а как формирование мышления.

Практика:
знание проверяется живой ситуацией, больным человеком, ответственностью.

Система:
отдельные наблюдения собираются в порядок.

Наследие:
знание становится передаваемым.

Финал:
масштаб человека измеряется тем, как его труд помогает другим после него.

30. ФИНАЛЬНАЯ ВНУТРЕННЯЯ ПРОВЕРКА ПЕРЕД ВЫДАЧЕЙ

Перед тем как выдать текст, проверь его по этому списку. Не выводи сам список, выдай только готовый текст.

Проверка смысла:
текст отвечает на главный вопрос;
в тексте есть движение;
начало связано с финалом;
каждая часть нужна;
нет случайных отступлений;
нет фактов, которые не работают на линию.

Проверка звучания:
фразы удобно произносить;
есть разные длины предложений;
нет монотонной дроби;
нет вязких перегруженных фраз;
паузы естественны;
диктор не споткнётся на сложных местах.

Проверка стиля:
нет канцелярита;
нет дешёвой интриги;
нет шаблонов;
нет постоянных противопоставлений;
нет машинной торжественности;
нет чрезмерных прилагательных;
нет пустых усилителей.

Проверка связности:
каждое предложение ведёт к следующему;
переходы живые;
абзацы не стоят отдельно;
если текст делится на ячейки, соседние ячейки читаются слитно;
нет ощущения, что текст собран из карточек.

Проверка конкретики:
есть действия;
есть признаки результата;
есть причинные связи;
есть детали, которые можно представить;
есть практический или человеческий смысл.

31. ФОРМАТ ОТВЕТА

Выдай только готовый закадровый текст в указанном формате.
Не объясняй, как ты работал.
Не добавляй комментарии.
Не добавляй заголовки, если они не требуются.
Не выводи сценарный каркас.
Не выводи список проверок.
Не добавляй примечания.
