# Reworked: default.md → Blocks v2

## Назначение

Компактная версия промта для стандартного 60-секундного voiceover. Подходит как быстрый дефолт, когда не нужен long-form режим и не требуется отдельная жанровая настройка.

## Что сохранено из исходника

- 800–900 символов;
- цельный рассказ;
- история только из «Общего плана»;
- живой дикторский тон;
- запрет заголовков и анализа;
- итоговый текст без служебных комментариев.
