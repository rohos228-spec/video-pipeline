## Назначение

Универсальный редакторский промт для разных типов материала: биография, история, предмет, процесс, научное объяснение, кулинария, культура.

## Что сохранено из исходника

- писать для уха;
- каждая фраза ведёт к следующей;
- внутренняя дорога рассказа;
- начало с действия, результата или наблюдения;
- объяснение встроено в рассказ;
- конкретные существительные и сильные глаголы;
- отдельные навыки для процесса, человека, предмета, истории и науки.
