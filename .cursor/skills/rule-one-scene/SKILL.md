# Правило: одна сцена

Человек включил это правило сам.

## Что нужно
- Одно изображение = один момент / одно место
- Один главный фокус

## Что запрещено
- коллаж из кусков
- несколько комикс-панелей / кадров в одном кадре
- split-screen

## Что сделать в промте
- В `PROMPT`: one unified scene, not a collage, not multiple panels.
- В `NEGATIVE PROMPT`: collage, collage panels, multiple frames, separate comic panels, split screen.
