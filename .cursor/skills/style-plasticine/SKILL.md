# Стиль: пластилин

Ты выбран человеком как **визуальный стиль**. Не спрашивай «какой стиль?» — он уже выбран.

## Что делать
1. Возьми описание сцены от человека (кто / где / что).
2. Собери один `PROMPT` в стиле ниже + `NEGATIVE PROMPT`.
3. Не выдумывай сюжет сверх его текста. Недостающее просто не добавляй.
4. Если включены `rule-*` скилы — соблюдай их.

## Вид
- Claymation / plasticine 2D-look, handcrafted miniature
- Мягкие округлые формы, fingerprints, matte clay
- Приглушённые земляные цвета, soft shadows
- Не vector, не фото, не glossy 3D, не неон

## Шаблон PROMPT
Create a single claymation-style 2D-look illustration of [сцена человека]. The scene should feel handmade, slightly melancholic but not scary, with a clear central narrative. Build everything from plasticine-like forms: soft rounded edges, visible fingerprints, matte texture, slight imperfections and simple stylized proportions. Use muted earthy colors (gray, dusty green, dark blue, beige, brown), soft shadows and a handcrafted miniature atmosphere. Keep one clear focal point. Final style lock: claymation / plasticine 2D-look, handcrafted miniature scene, soft rounded edges, fingerprints, matte texture, slight imperfections, muted earthy colors, soft shadows.

## NEGATIVE (база)
text, letters, numbers, logos, readable signs, sharp digital vector art, photorealism, glossy 3D render, realistic human anatomy, bright neon colors, clutter, low detail, blurry, watermark, frame

```text
PROMPT:
…

…
```
