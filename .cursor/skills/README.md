# Скилы для image prompt — выбираешь сам

Это не «чат с ИИ по шагам». Ты сам включаешь нужные скилы в Cursor и пишешь сцену.

## Как собрать комплект

1. Включи **один** стиль (`style-*`).
2. По желанию включи правила (`rule-*`).
3. Напиши сцену своими словами (кто / где / что происходит).
4. Получи `PROMPT` + `NEGATIVE PROMPT`.

Пример комплекта: `style-polka` + `rule-no-text` + `rule-one-scene` + `rule-9x16`.

## Стили (выбери один)

| Скил | Что получишь |
|------|----------------|
| `style-plasticine` | Пластилин / claymation, мягкий handmade |
| `style-knitted` | Вязаный / войлок / детская textile-иллюстрация |
| `style-noir` | Грязный нуар, true-crime постер |
| `style-polka` | Trash Polka: grunge + красные акценты |

## Правила (можно несколько)

| Скил | Что делает |
|------|------------|
| `rule-no-text` | На картинке нельзя читать буквы/цифры/лого |
| `rule-one-scene` | Один кадр, не коллаж и не комикс-панели |
| `rule-no-gore` | Без явной крови и жести |
| `rule-9x16` | Вертикальный кадр 9:16 |

## Важно

- Сюжет берётся **только** из твоего текста. Скил не придумывает историю за тебя.
- Два стиля сразу не включай — будет каша.
